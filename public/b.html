C · 피팅 표준화 조정 강제 · A4 피팅시트 출력
입력 측정 5값 + 안면각 + 소재 ｜ 앱 표준→예비→정밀 3단계, 휜 값·조정 기록 → 본부 DB
요소기준·표준치이상치 → 처리
다리벌림각〔확인〕 목표 템플간격 = 귀윗간격 −15(유연금속)/−10(반강성)mm계수 현장 검증
판토스코픽 틸트〔확인〕 8~12°〔확인〕 8~12° 채택 (중앙 10°)
정점간거리〔확인〕 12mm, 고도수일수록 짧게편차 → 코받침·틸트 조정
안면각/랩각〔확인〕 안면각 5°→BC4 / 10°→BC6 / 15°+→랩 전용고도수 왜곡 → 보정
코받침코 형태 기반, 수직 안착+정점거리일체형 브리지 → 조정 한계
템플 꺾임·귀뒤귀 위치/높이, 수평 유지미끄럼 → 재조정
소재→최대벌림각〔확인〕 티탄±8 / 금속±6 / TR±4 / 경질아세테이트±1 mm초과 → "이 테 못 맞춤" 안내

A4 피팅시트 = 출력물. 각도를 종이에 직접 안 재고 선형값(mm)으로 환산해 잰다. 평면도에 목표 템플간격 두 점 → 거기 닿게 휨. 측면도엔 판토 각도 웨지. 50mm 스케일바(가로·세로) 필수, 인쇄 "실제 크기 100%". 역할 3: 휨 지그 + 검증 게이지 + 고객 설명. (구현체 = bv-fitsheet-demo.pdf)

D · 안전·의뢰 기준 안경사법 경계 — 매뉴얼의 안전장치

안경사는 도수 안경·콘택트의 조제·판매·피팅이 업무. 질환의 진단·치료·처방은 안과 영역. 이 경계를 시스템이 자동으로 지킨다 — "이상치 → 의뢰" 트리거가 그 구현.

트리거기준처리
연령 제한6세 이하 = 의사 처방 필수 (의료기사법 시행령 별표1) — 안경사 단독 조제·판매 불가해당 시 → 안과 처방 안내
질환 의심〔확인〕 급격 시력저하·복시·시야결손·동공부등·충혈통증·번쩍임/날파리(망막)진단·치료 금지 → 안과 의뢰
교정 불충분최대교정에도 목표 미달→ 의뢰
양안시 이상억제·복시·큰 사위→ 프리즘 검토/의뢰

"데이터 장악"과 "법적 안전"을 동시에. 의뢰조차 기록되면 그것도 본부 데이터.

E · 채울 칸 인덱스 임상 작업 리스트

아래만 채우면 v0.1 골격이 v1.0 정본이 된다. 〔확인〕은 검증·확정, 〔＿＿〕은 신규 입력.

  • 검안 컷값 — 나안·교정시력 정밀 기준, 고도수·고난시 주의 컷, 구도수 대비 허용 변화폭.
  • 양안시 정상범위 — 사위(원/근) Morgan 채택 여부, NPC·AMP·AC/A 기준, 억제·복시 의뢰 컷.
  • ADD 연령 기준 확정(작업거리 반영).
  • 조제 계수 — 정점거리 보정 적용 컷, 인셋 산출식, 굴절률 선택 기준.
  • 피팅 계수 — 다리벌림각 −15/−10 검증, 판토 8~12 vs 8~15, 안면각→베이스커브 매칭.
  • 소재→최대벌림각 테이블(②와 공유) 수치.
  • 안전·의뢰 — 6세 이하 법규 확인, 질환 의심 소견 목록 확정.
