업셀 스크립트
컴플레인 대응
원칙: 강매 금지. 측정·검안 데이터에 근거한 제안만. "권유"가 아니라 "당신 데이터를 보니 이게 맞다".
렌즈 업그레이드 업셀
상황
고도수(-4.00 이상)인데 저굴절률 선택 시
멘트
"도수가 있으셔서 1.60으로 하면 렌즈가 두꺼워 테 밖으로 나와요. 1.67로 가면 두께가 30% 얇아져서 훨씬 가볍고 보기 좋습니다."
근거
검안 도수 → 굴절률 권장 자동 표시(rx 연동). 미적·무게 실측 차이.
청광·변색 코팅 업셀
상황
용도가 업무·VDT(모니터)거나 야외 활동 많을 때
멘트
"하루 종일 모니터 보신다고 하셨죠. 청광 차단 넣으면 눈 피로가 덜해요. 야외도 잦으면 변색렌즈 하나로 선글라스까지 됩니다."
근거
접객 카드의 '용도' 필드 → 코팅 매칭. 라이프스타일 기반.
세컨드 안경 / 선글라스 업셀
상황
단골·재방문, 이미 측정 데이터 보유
멘트
"사이즈가 이미 저장돼 있어서 두 번째 안경은 측정 없이 바로 됩니다. 운전용 변색이나 도수 선글라스 하나 더 어떠세요?"
근거
통합 프로필에 9사이즈·도수 저장 → 추가 구매 마찰 0. 재구매 훅.
콘택트 병행 업셀
상황
운동·외출 시 안경 불편 언급
멘트
"운동할 때 안경이 불편하시면, 1day 콘택트를 같이 쓰시는 분들 많아요. 도수는 이미 알고 있으니 맞는 제품 안내드릴게요."
근거
검안 도수 → 콘택트 도수 환산. 카탈로그 가격 투명.
원칙: 고객 잘못으로 몰지 않는다. 1차 매장 즉시 대응 → 안 되면 본사 에스컬레이션. 모든 건 로그 기록.
고객 접수→매장 1차 대응→본사 에스컬레이션→보상·처리→로그 기록
도수가 안 맞음 1차 매장
대응
즉시 재검안(무료). 적응 기간(1~2주) 안내. 처방 오류 확인 시 렌즈 무상 재제작.
멘트
"불편하셨겠어요. 바로 다시 검사해드릴게요. 검사 결과 도수가 다르면 렌즈는 무상으로 다시 만들어 드립니다."
기준
검안 오류 = 본사 비용 무상 재제작. 적응 문제 = 재검 후 안내.
피팅 불편 (흘러내림·압박) 1차 매장
대응
즉시 재피팅(무료·무제한). A4 피팅지 재측정 → 벌림각·코받침 재조정.
멘트
"피팅은 언제든 무료로 다시 맞춰드려요. 지금 바로 조정할게요."
기준
피팅은 영구 무상. 측정 데이터 재활용.
파손 (힌지·렌즈) 본사 연계
대응
무상 A/S 기간 내 = 무상 교체. 기간 외 = 본부 표준가 부품 교체. 자가 파손도 부품가만.
기준
A/S 기간: 프레임 1년. 부품 본부 공급가 투명 안내.
환불 요구 본사 연계
대응
제작 전 = 전액. 제작 후 단순변심 = 정책 안내(맞춤 제작 특성). 하자 = 전액 환불·재제작 택1.
기준
맞춤 제작품 환불 정책 사전 고지(주문서 동의). 하자는 무조건 구제.
제작 지연 1차 매장
대응
지연 사유·새 일정 즉시 알림톡. 약속 초과 시 소정 보상(쿠폰).
기준
알림톡 자동 발송(준비완료 트리거 연동).
에스컬레이션 기준 본사
등급처리권한
L1 매장재검·재피팅·일정 안내가맹점 즉시
L2 본사 연계무상 재제작·부품·환불본사 승인
L3 본사 직접분쟁·보상·법적본사 CS 전담
로그
모든 컴플레인은 고객 프로필에 기록 → 반복 유형 본사 분석 → 제품·매뉴얼 개선.
원칙(고객 2): 무관심도, 과잉 접객도 아니게. 고객은 따라다니는 걸 싫어하지만 완전 방치도 뻘쭘하다. 신호를 읽고 타이밍에 들어간다.
입장 0~30초 방치·추적 금지
행동
가벼운 인사 한 번. "편하게 둘러보세요." 그 후 1.5m 거리 유지, 따라다니지 않기.
금지
첫 30초 영업 멘트 · 가격부터 꺼내기 · 뒤를 졸졸 따라다니기
신호 포착 접근 타이밍
신호
같은 안경 2회+ 만짐 · 거울 2회+ 봄 · 두 개 비교 · 두리번 · 눈 마주침
멘트
"그 모델 마음에 드세요? 그거 9사이즈로 나와서 얼굴에 맞춰 드릴 수 있어요. 써보실래요?"
근거
강요가 아니라 정보. 우리 강점(9사이즈·측정)을 자연스럽게.
신규 vs 단골 강도 분기
신규
여유 있게. 측정·가상피팅 권유로 부담 낮추기. "온라인서 미리 골라보셔도 돼요."
단골
프로필 호출 → 바로 데이터 기반. "지난 사이즈 그대로 있어서 바로 보여드릴게요."
이탈위험
부담 없이 안부 + 무료 재검 제안. 영업 톤 자제.
접객 데스크(counter)가 조회 시 접객 강도를 자동 표시 — 단골/VIP/이탈위험별 톤을 띄워준다.
