맞춤안경을
프랜차이즈로 복제한다

측정·모듈·매뉴얼로 맞춤을 표준화하는 프리미엄 안경 프랜차이즈 ｜ 사업 개요 ｜ 2026.06
핵심 한 줄. 3D프린터가 못 하는 걸 우리가 한다 — 맞춤을 '출력'이 아니라 '측정·모듈·매뉴얼'로 분해해서 복제 가능하게 만든다.
1문제 — 시장의 빈칸 2왜 3D프린터가 답이 아닌가 3우리 대안 — 맞춤의 분해 49사이즈 — 재고를 안 터뜨리는 구조 5검안·맞춤·데이터 장악 — 누가 뭘 얻나 ★ 6매뉴얼 = 데이터 장악 장치 7미래 스마트안경 시장에서도 먹히는 이유 ★ 8그래서 못 따라잡힌다 — 해자

1.문제 — 시장의 빈칸

안경 시장은 세 갈래인데, 가운데가 비어 있다.

  • 맞춤은 비싸다. 3D프린팅 맞춤안경(브리즘)은 객단가 약 42만 — 한 명당 한 벌을 깎아 출력하는 구조라 대중가가 불가능하다.
  • 싼 건 안 맞는다. 대형 체인(다비치)·일반 기성테는 사실상 원사이즈여서, 한국인 얼굴의 다양성(폭·코·귀 위치)을 받아내지 못한다.
  • 스마트안경은 한국인 얼굴·도수가 안 된다. 빅테크 제품 = 원사이즈 + 도수 미지원. 멋있어도 못 쓰는 사람이 대다수.
빈칸

대중가로 맞고, 도수 되고, 확장 가능한 안경. 이 자리가 비어 있다. 이게 우리 자리.

2.왜 3D프린터가 답이 아닌가

"맞춤 = 3D프린팅"이 미래라는 통념. 우리는 여기에 동의하지 않는다.

  • 소재 한계. SLS 나일론은 아세테이트·티탄의 손맛·광택·내구·수리성을 못 따라감. 안경은 패션재인데 프린팅 질감이 티난다.
  • 단가·속도. 1개 출력 + 후가공(염색·연마)에 시간·비용 폭증 → 객단가 42만이어야 수지가 맞음 → 대중화 구조적으로 불가.
  • 확장 불가. "맞춤 = 출력"은 한 명에 한 출력이 묶이는 1대1 종속. 프랜차이즈로 복제가 안 된다.
  • 수리 불가. 부러지면 그 사람 데이터로 또 출력. 모듈 교체 개념이 없다.
우리 판단

미래는 프린터가 소재를 대체하는 게 아니라, 검증된 소재(아세테이트·티탄)를 '프린터처럼' 맞춤으로 운용하는 것. 그 운용법이 우리의 모듈식 9사이즈 + 측정→피팅 지시다.

3.우리 대안 — 맞춤의 분해

맞춤을 통째로 출력하지 않는다. 세 조각으로 쪼갠다.

맞춤 = 측정(SW) + 모듈(기성 소재) + 피팅지시(매뉴얼)

세 조각 모두 복제 가능하다. SW는 깔면 되고, 모듈은 본부가 공급하고, 매뉴얼은 가맹점에 주면 된다. 출력은 복제가 안 되지만 이 셋은 된다.

팀 설득 비유 (한 줄로 외울 것)

브리즘 = 한 명당 한 벌 깎는 맞춤 양복점(비싸고 느림).
우리 = 측정해서 기성 모듈을 조합·수선하는 맞춤 = 유니클로 + 수선집.
둘 다 "맞춤"인데, 우리만 대중가 + 확장 가능.

요지: 3D프린터가 못 하는 건 '확장'이고, 우리가 가진 건 정확히 '확장'이다.

4.9사이즈 — 재고를 안 터뜨리는 구조

"맞춤이면 재고 터지는 거 아니냐"는 팀의 첫 의심. 안 터진다. 세 층이 다르기 때문.

층정체숫자
9사이즈 (핏 등급)측정값이 매핑되는 분류. 프론트폭 {S·M·L} × 템플길이 {S·M·L}3×3 = 9
810 (모듈 카탈로그)프론트·템플을 따로 소싱해 나사 조립. 프론트모듈 × 템플모듈30×27 = 810
수요표 (재고 통제)9등급별 예상 판매 비중 → 생산량·매장 샘플 배치M·M 최대

810은 카탈로그 숫자지 진열 숫자가 아니다. 매장엔 수요표 가운데(M·M) 위주 대표 샘플만 깔고, 실판매는 측정 → 9등급 매칭 → 본부 발주·조립(브리즘 준맞춤과 같은 구조). 그래서 선반에 810개가 안 쌓인다.

모듈식의 진짜 무기

프론트·템플이 따로 조립되니 '프론트 L + 템플 S' 같은 독립 조합이 된다. 기성테는 둘이 한 몸이라 "얼굴 넓은데 귀까진 짧은 사람"을 못 맞춘다 — 우리는 맞춘다. 이게 9사이즈를 두 독립 축으로 굴리는 이유이자 맞춤 포지션의 핵심.

5.검안·맞춤·데이터 장악 — 누가 뭘 얻나 ★

같은 시스템이 세 주체에게 각각 다른 이득을 준다. 이게 설득의 본체.

소비자가 얻는 것
광학적으로 맞고(측정·검안) + 패셔너블(디자인·큐레이션) + 내 데이터 기반(얼굴·이력). 추천이 감이 아니라 근거 — "이 젠틀몬스터 78% vs 우리 PB(자체 브랜드) 96%"처럼 적합도를 숫자로 비교하니, 강매 없이 신뢰가 생긴다. 재구매 땐 내 데이터로 더 빠르고 정확하게.
가맹점주가 얻는 것
검안·발주·정산·마케팅이 자동화 → 안경사 1명 + 일반 직원으로 운영. 추천·가격이 매뉴얼화돼 도제식("선배 눈치껏") 스트레스에서 해방. 기존 브랜드 팔던 방식 그대로 + 측정·추천으로 객단가·전환율↑. 진입장벽↓.
본사(우리)가 얻는 것 = 장악
타사 거래까지 포함해 모든 거래가 우리 데이터로 귀속(데이터 본부소유). 프론트·템플·렌즈 공급 독점. 매뉴얼 자체가 데이터 수집기 — 가맹이 늘수록 데이터·공급 지배력이 커진다.

그래서 우리가 추구하는 것 / 장악하는 것

추구한다 (어떻게)장악한다 (무엇을)
자동화 — 아무나 운영
맞춤화 — 측정→매칭→피팅
표준화 — 안경사 편차 제거		 데이터 — 얼굴·구매·만족 누적
공급 — 프론트·템플·렌즈 독점
표준 IP — 측정·검안 경계값·매뉴얼

자동화·맞춤화·표준화는 '수단'이고, 데이터·공급·IP 장악이 '목적'이다. 수단이 곧 데이터 수집 경로가 되도록 설계한 게 핵심.

6.매뉴얼 = 데이터 장악 장치

안경학과 시험범위 그대로 — 검안 → 사이즈 재기 → 조제 → 피팅까지 표준 절차를 앱이 단계별로 띄운다. 따라가면 자동으로 표준 준수.

  • 비숙련도 베테랑급(목표a) + 숙련자 판단차 제거(목표b), 둘 다.
  • AI가 기준값·이상치·다음 단계 제시 → 안경사는 확인·확정만.
  • 가맹점이 매뉴얼대로 일하면 데이터가 자동으로 본부로 귀속. "쉽게 주고 데이터 장악"이 정확히 이 구조.
검안 → 측정(5값) → 객관화 추천 → 본부 표준가 → A4 피팅시트 → 정산·발주 → 재구매 마케팅 (전 과정 자동·표준)

이 매뉴얼이 곧 마스터 설계도의 ③(검안·피팅 표준화)이자 가맹 패키지의 본체다.

7.미래 스마트안경 시장에서도 먹히는 이유 ★

우리는 원래 스마트안경(오디오글라스)으로 시작했다. 그 목표는 버린 게 아니라, 더 좋은 자리에서 다시 잡는 것.

과거 패착 → 전략 수정

패착 = 처음부터 돈 들여 하드웨어 설계부터 하려 한 것. 수정 = 프차로 먼저 수익·인프라·데이터·유통망을 깔고, 병목에서 기다린다. 시장 열릴 때 시작해도 늦지 않다.

스마트안경 3대 약점 → 우리가 그대로 커버

  • 안 예쁨 → 파라메트릭 9사이즈 + 복각 디자인 + 큐레이션.
  • 원 사이즈(안 맞음) → 9사이즈 + 자동 측정·피팅 = 한국인 얼굴 맞춤.
  • 도수 안 됨 → 자동 검안 + 도수 안경 가능 = 시력 교정까지.
병목 포지션 + 모듈식의 구조적 이점

빅테크가 하드웨어를 만들어도 "한국인 얼굴에 맞고 도수 되는 안경으로 만들어 파는" 마지막 단계(측정·피팅·도수·유통)는 우리 같은 플레이어가 필요. 미리 갖춰 두면 시장 열릴 때 병목에서 기다리면 된다. (메타가 안경 부분은 룩소티카에 맡긴 것처럼, 빅테크의 한국 안경 파트너 자리를 미리 선점)

여기서 브리즘은 못 한다. 풀출력 구조라 배터리·기판·스피커가 들어가는 스마트안경을 못 만든다 — 결국 표준 모듈을 따로 생산해 보조 맞추는 수밖에 없고, 그 순간 "100% 맞춤" 정체성이 깨진다. 우리는 처음부터 모듈식이라 '프론트 = 전자모듈 / 템플 = 배터리·스피커'가 구조적으로 자연스럽다.

결론: 처음 목표(스마트안경)로 돌아오는데, 이번엔 인프라 깔고 병목 잡은 채로 돌아온다.
