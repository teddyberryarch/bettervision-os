성수 안테나샵 기획

도수를 하지 않는 브랜딩·체험 매장. 안경사 면허 없이 지금 열 수 있는 첫 거점.

핵심 — 면허 없이 가능. 안경사 면허는 시력보정용(도수) 안경의 조제·판매에만 필요하다. 선글라스·스마트(오디오)안경·비도수 패션안경·저도수 리딩글라스는 면허 없이 누구나 판매 가능. 안테나샵은 도수를 하지 않으므로 검안·가공 기계도 면허도 불필요.

1. 역할 — 재고 매장이 아니라 체험·브랜딩·데이터

  • 브랜드 각인 — SAGA 인테리어 + LED 가변 설치(본부가 OS로 콘텐츠 푸시). 성수 상권에서 "이런 안경 브랜드가 있다"
  • 측정 체험 — 카메라로 얼굴 측정 → 9사이즈·가상피팅을 보여주는 무대(도수 아님, 측정·추천만)
  • 판매 — 선글라스 · 오디오(스마트)안경 · 비도수 패션 = 면허 무관 품목. 온라인몰로 연결
  • 데이터 수집 — 측정·관심 데이터 본부 귀속(밸류업 근거)
  • 도수 손님 — 제휴/본점(로마안경원) 또는 향후 도수 매장으로 연계

2. 구성 · 장비 (가벼움)

항목내용
측정아이패드 카메라 측정 1대 (얼굴 5값·가상피팅)
검안·가공 기계없음 (도수 안 함)
피팅 수공구기본 세트(워머·플라이어·세척)
진열대표 모델 견본 + 선글라스·오디오안경. 곱셈을 덧셈으로(측정→매칭)
인테리어SAGA 톤 + LED 포컬. 빈 공간·조명 집중·강한 포컬 1개

초기 장비 비용 대략 200~300만 원 + 인테리어. 도수 매장(3,500~6,500만) 대비 진입 부담이 가장 가볍다. 〔견적 확인〕

3. 운영

  • 인원 1~2인 (면허 불필요). 측정→가상피팅→선글라스/오디오 판매·온라인 유도
  • 온라인몰 + 앱 연계(비도수는 온라인 판매 제한 없음)
  • 본부 OS로 가격·콘텐츠·데이터 통제

4. 펀딩·로드맵 연결

  1. 자비로 안테나샵 오픈(면허 전, 지금)
  2. 6개월 데이터 + OS 가동 → 밸류업
  3. 초기창업패키지(2027 Q2 최종창) / 투자 유치
  4. 도수 매장(신규·전환)은 면허(2026.12) 또는 종사 안경사 고용 후 확장
시간 가치. 12월 면허 전 6개월을 "대기"가 아니라 "안테나샵으로 브랜드·측정·데이터 인프라 까는" 기간으로. 병목에서 기다리는 게 아니라, 기다리며 인프라를 깐다.

비도수 판매·온라인 판매의 구체 범위는 품목별로 안경사법·전자상거래법 확인 권장. 도수 관련은 면허/제휴 필요.
